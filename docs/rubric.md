# Insight Quality Rubric

Score each of the top-10 insights 1–5 on each dimension. Ideally have a
second rater score independently and report mean + disagreements.

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| **Groundedness** | Claims not supported by quotes | Mostly supported, minor stretch | Every claim traceable to cited quotes |
| **Actionability** | Vague ("improve quality") | Directionally useful | Team could start next sprint |
| **Root-cause plausibility** | Speculative/wrong | Plausible but generic | Specific, testable hypothesis |
| **Evidence sufficiency** | Single/duplicate voices | Some breadth | Multiple sources, time-spread, n adequate |
| **Clarity** | Confusing | Readable | Kanban-card ready |
